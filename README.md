# Sistema de Gestión C++

Proyecto básico de **sistema de gestión de inventario** desarrollado en C++.

## Archivos importantes
- `Main.cpp` – Código principal
- `inventario.txt` – Archivo de inventario
- `reporte.csv` – Archivo de reportes
- `sistema_gestion_marketplace.cbp`, `.depend`, `.layout` – Archivos de proyecto de C++ Builder
- `.gitignore` – Archivos que no se suben (compilación, temporales)

## Uso
1. Abrir `Main.cpp` en tu IDE de C++ preferido.
2. Compilar y ejecutar el programa.
3. `inventario.txt` y `reporte.csv` se usan para probar la funcionalidad básica del sistema.

## Descripción
Este proyecto es un ejemplo funcional de manejo de archivos y estructuras básicas en C++, ideal para demostrar conocimientos básicos de programación y gestión de datos en un portafolio.
